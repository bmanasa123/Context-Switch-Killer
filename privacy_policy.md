# Privacy Policy for Context Switch Killer

Last updated: March 27, 2026

## Overview

Context Switch Killer is a Chrome extension that helps users save, categorize, and restore browser tab sessions.

## What Data Is Stored

The extension stores the following information locally on the user's device using `chrome.storage.local`:

- Session names entered by the user
- Tab titles
- Tab URLs
- Session category labels such as Coding, Learning, Shopping, or General
- Session timestamps and tab counts

## How Data Is Used

This data is used only to:

- Save the current browser session
- Show saved sessions inside the extension popup
- Restore saved tabs when the user chooses a session
- Delete sessions when requested by the user

## Data Sharing

Context Switch Killer does not send, sell, transfer, or share user data with any third party.

## Remote Servers

The extension does not transmit browsing data to any remote server.

## Data Retention

Saved sessions remain on the user's device until the user deletes them or removes the extension.

## User Control

Users can:

- Create new saved sessions
- Restore previously saved sessions
- Delete saved sessions at any time
- Remove the extension to delete locally stored extension data
